# Changelog

All notable changes to `nprobust` will be documented in this file.

## 1.0.0 - 2026-05-17

- Initial coordinated release of the R, Python, and Stata implementations.
- Added kernel density estimation and bandwidth selection through `kdrobust` and `kdbwselect`.
- Added local polynomial regression estimation and bandwidth selection through `lprobust` and `lpbwselect`.
- Included illustration scripts and example data for R, Python, and Stata.
